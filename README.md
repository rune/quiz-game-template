# Template for a Quiz Game

Put the theme's questions, style sheet and name file into a directory under `themes`

You can use `npm run theme <theme-name>` or `yarn theme <theme-name>` to switch between themes.

# Build and Test

```
yarn
yarn build
yarn dev
```
